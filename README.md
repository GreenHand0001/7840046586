# 项目简介

## 核心功能
- 基于 YOLOv8 的考场行为检测系统，已训练完成可直接使用
- 支持对图片、视频及摄像头进行实时检测
- 支持 14 类行为识别：正确姿势、右移、左移、后移、站立、前移、注视、无聊、询问、鞠躬、手部动作、低头、作弊、窥视
- 提供交互式界面，实时显示目标位置、目标总数及置信度信息
- 支持批量检测并查看所有检测结果
- 提供检测结果保存功能
- 支持自定义置信度和 IOU 阈值
- 可选择显示检测标签和原图
- 支持多种检测模型选择

## 适用场景
- 考场监控与行为分析
- 教育机构安全管理
- 考试监考自动化
- 行为研究与分析
- 视频内容审核

## 技术特点
- 基于 Python 3.9 开发
- 使用 OpenCV-Python、PyQt5、Ultralytics、Torch1.9 等技术栈
- 采用 YOLOv8 作为核心检测模型
- 提供完整的用户界面，支持多种输入源
- 包含 5000 张标注数据集，确保模型准确性
- 已配置好运行环境，部署简单快捷
